# Hues of Memory

An interactive web installation exploring the chromatic essence of human memory through randomized sensory mapping.

## Project Overview
*Hues of Memory* is a digital-urban intervention based in **Mexico City (CDMX)**. Physical QR codes are placed in public spaces, inviting passersby to discover a fleeting connection between a specific color and a localized memory. 

Each scan triggers a unique, randomized experience where the screen is flooded with a single hue accompanied by a brief, evocative text in Spanish, reflecting the subjective nature of our past.

## Features
- **Generative Experience:** Every page load delivers a new color-memory pairing.
- **Minimalist Design:** Immersive full-screen transitions focused on color theory.
- **Mobile-First:** Optimized for instant loading and seamless QR interaction.
- **Bilingual Context:** Professional documentation in English with localized content in Spanish.

## Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Properties & Transitions).
- **Logic:** Vanilla JavaScript (ES6+).
- **Data:** Structured JSON for memory mapping.
